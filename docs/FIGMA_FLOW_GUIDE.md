# Figma Flow Diagram Guide

This project includes a ready-to-import flow diagram source for Figma.

## Files
- docs/project-flow-figma.mmd: Mermaid flow diagram source

## How to view in Figma
1. Open your Figma file (or FigJam board).
2. Install a Mermaid plugin in Figma (for example: Mermaid Chart).
3. Open docs/project-flow-figma.mmd and copy all text.
4. In the Figma Mermaid plugin, paste the text and generate.
5. Share the Figma link with your team.

## Audience notes
- The diagram is grouped for business readability:
  - Frontend Experience
  - Backend API Engine
  - Storage and External
  - Delivery and Governance
- Arrows show end-to-end flow from user action to data storage and reporting.
